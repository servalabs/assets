# Video

Downloadable recordings and captures for the in-development gallery site (`apps/website-next`).

| File             | Format         | Description                                             |
| ---------------- | -------------- | ------------------------------------------------------- |
| `demo.mp4`       | 16:9 landscape | Clean rendering of the experience, no background effect |
| `stage.mp4`      | 16:9 landscape | Blurred backdrop fill behind the experience             |
| `reel.mp4`       | 9:16 portrait  | Vertical format with blurred backdrop                   |
| `screenshot.jpg` | still          | Static capture of the experience                        |

To download from GitHub: open the file → **Download raw file** (top right).
